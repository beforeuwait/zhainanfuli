## 更新

2020-2-18:

发现用代理下载图片的过程中，会出现图片不完整
    
同时这种情况还很多
    
随即更新了一版

    通过二进制图片结尾 '\xd9' 来判断
    下载过程中，判断该图片是否为完整图片
    同时对于已经下载的图片仍旧判断是否为完整图片 不完整则重新下载
    牺牲了一定的性能，但是保证完整性

## 启动方式

    下载种子:
    python3 download_seeds.py
    
    自动遍历该网站所有分类，然后遍历各个分类里具体的图集，并保存下url
    
    下载图片:
    默认是 10个进程下载，需要的话，自己进代码里修改
    
    python3 download_images.py
    
    说明, 图片链接我已经采集好了，不用再下载种子了，直接下载图片就行了
